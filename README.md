# 🍕 Pizza Sales Data Analysis & Dashboard

## 📌 Project Overview

This project analyzes pizza sales data to uncover business insights such as revenue trends, best-selling pizzas, and customer ordering behavior.

It includes both:

* 📊 Power BI Dashboard
* 🐍 Python-based Exploratory Data Analysis (EDA)

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Power BI
* Excel / CSV Data

---

## 📂 Project Structure

* **data/** → Raw dataset
* **notebooks/** → Python analysis (EDA)
* **dashboard/** → Power BI dashboard
* **images/** → Dashboard screenshots

---

## 📊 Key Business Insights

* Peak order time occurs during evening hours
* Classic pizzas generate the highest revenue
* Large size pizzas are most preferred
* Weekend sales are higher than weekdays
